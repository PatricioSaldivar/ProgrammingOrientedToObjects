Factorial


Descripcion: Programa que tiene una funcion para calcular el factorial de un numero
Autor: Yolanda Martinez
Fecha: 28 de Enero de 2017

Funcion: factorial


Descripcion: Calcula el factorial de un número
Parametro: iNum valor para el que se va a calcular el factorial. Debe ser entero positivo.
Retorna: factorial del valor iNum

Funcion: validaEntero


Descripcion: Función que pide al usuario un valor y valida que sea entre 1 y 10
Retorna: regresa el valor entero que tecleó el usuario (entre 1 y 10)

Funcion: main


Descripcion: Pide el valor al usuario, calcula el factorial y lo muestra
